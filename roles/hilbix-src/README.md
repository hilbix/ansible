# `hilbix-src`

```
  roles:
    - role: hilbix-src
      name: Your Name
      email: your@mail
```

# Description

Install <https://github.com/hilbix/src> and associated standard tools

## Parameters

- `gitname`: The name in the standard `.gitconfig` (no default)
- `gitmail`: The email in the standard `.gitconfig` (no default)
- `getdefault`: The default branch (defaults to "master")

## Not yet implemented

- `github`: Your GitHub name

## Bugs

- Not really configurable nor autodetecting
