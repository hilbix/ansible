# `hilbix-src`

```
  roles:
    - role: hilbix-src
      name: Your Name
      email: your@mail
```

# Description

Install <https://github.com/hilbix/src> and associated standard tools

## Parameters

- `name`:  The name in the standard `.gitconfig`
- `email`:  The email in the standard `.gitconfig`

## Bugs

- Not really configurable nor autodetecting
